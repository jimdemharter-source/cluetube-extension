# ClueTube Landing Page & API

This repository hosts the official landing page and the backend API for the **ClueTube Chrome Extension**.

## 🚀 Get Started

Visit the live landing page to download the latest version of the extension:
**[cluetube-extension.vercel.app](https://cluetube-extension.vercel.app)**

## 📦 Installation guide

1. **Download**: Click "Download" on the landing page to get `cluetube-extension.zip`.
2. **Extract**: Unzip the file to a folder on your computer.
3. **Chrome Extensions**: Open `chrome://extensions` in your browser.
4. **Developer Mode**: Toggle "Developer mode" in the top right.
5. **Load Unpacked**: Click "Load unpacked" and select the extension folder.

## 🛠️ Technical details

- **Frontend**: Single-page static site (`index.html`) optimized for Vercel.
- **Backend**: Python-based serverless functions (`api/index.py`).
- **Runtime**: `@vercel/python`.
- **Packaging**: Standardized ZIP format for unpacked installation.
